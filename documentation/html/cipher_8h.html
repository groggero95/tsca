◆ BLINDING

#define BLINDING   0

Enable blinding in the timing acquisition on the OS.

Used in the file timing.c to selct between normal modular exponentiation and a blinded version of it.

  • 1 Select the ME_big_blind function as target for measurement
  • 0 Select the ME_big function as target for measurement

◆ VERSION

#define VERSION   0

Select one of the predifined keypairs.

For each size of integer there are some predefined keypair which can be used for time acquisition on the OS and for attacking their relative samples.

Remarks
Even if some keypair are available over the 256 bit mark no sample data is available for them
